# Weather app Kotlin

A Weather app which uses openWeatherMaps api and Google Maps api.  
 It uses Pojo classes to retrieve data from json.
 
 ## Libraries Used
    Gson Converter
    
 ## Gradle Version
    6.1.1
    
 ## Android Studio Version
    4.0
